---
이 소프트웨어는 'XE 게시판'모듈의 스킨으로 무료 제공스킨입니다.

이 소프트웨어는 GPL GPL을 채택한 오픈 소스 소프트웨어(OSS)로써 자유롭게 사용할 수 있는 자유 소프트웨어입니다. 이 라이선스의 조건을 준수하는 조건하에 누구나 자유롭게 사용할 수 있으며 개작 할 수 있습니다.
